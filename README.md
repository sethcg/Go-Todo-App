## Setup:

Clone Repository:
```
git clone https://github.com/sethcg/Go-Todo-App.git ./go-todo-app
cd ./go-todo-app
```

```
ren .env.example .env
```

Build and Run:
```
docker build -t go-todo-app .
docker run -p 3000:3000 go-todo-app
```
